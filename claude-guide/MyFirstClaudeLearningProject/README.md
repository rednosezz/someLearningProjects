# MyFirstClaudeLearningProject

我的第一个 Claude 使用学习项目

## 项目简介

这是一个用于学习 Claude Code 的 Java 入门项目。

## 使用的 AI 模型

- **AI 助手**: Claude Code
- **模型**: Claude 4.6 (qwen3.5-plus)

## 项目结构

```
MyFirstClaudeLearningProject/
├── src/
│   └── com/
│       └── example/
│           └── HelloWorld.java
├── .gitignore
└── README.md
```

## 编译和运行

```bash
# 编译
javac src/com/example/HelloWorld.java

# 运行
java -cp src com.example.HelloWorld
```
